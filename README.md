# Run using Docker 

```
$ git clone 
$ cd repo
$ export JEKYLL_VERSION=3.8
$ docker run --rm --volume="$PWD" -it jekyll/jekyll:$JEKYLL_VERSION jekyll build
$ jekyll serve
```
